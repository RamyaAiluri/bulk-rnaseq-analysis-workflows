# Bulk RNA-seq Workflow Using Bowtie2

This folder contains a basic bulk RNA-seq workflow for paired-end sequencing data.

The workflow performs:

1. Quality control of raw FASTQ files using FastQC
2. A combined raw-read quality report using MultiQC
3. Adapter and quality trimming using Trim Galore
4. Quality control of trimmed FASTQ files
5. A combined trimmed-read quality report using MultiQC
6. Read alignment using Bowtie2
7. SAM-to-BAM conversion and sorting using SAMtools
8. BAM file indexing
9. Gene-level read counting using featureCounts

## Bowtie2 Index

This workflow uses an existing Bowtie2 genome index.

The Bowtie2 index is not generated by this script. Update the `BOWTIE2_INDEX` variable with the prefix of a previously generated Bowtie2 index.
